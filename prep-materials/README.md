# NVIDIA NCA-GENL 两个月备考包

适用对象：汽车机械行业背景，做过机器学习预测模型和 AI 学习平台，非传统 IT 岗，但对 AI 有持续兴趣，并计划用约两个月准备 **NVIDIA-Certified Associate: Generative AI LLMs (NCA-GENL)**。

本资料以 NVIDIA 中文官网和官方考试学习指南为主，并结合你的背景做了取舍：少走“从零学编程”的路线，多补“考试会问、但日常 vibe coding 容易跳过的底层概念、NVIDIA 工具栈和实验评估”。

## 总体打法

1. 前 3 周补齐 LLM 基础、Transformer、RAG、Prompt Engineering。
2. 第 4-6 周把重点转到考试高权重领域：实验评估、软件开发、NVIDIA 工具栈。
3. 第 7 周处理 Trustworthy AI、隐私、偏见、幻觉、治理。
4. 第 8 周只做三件事：刷题、错题复盘、记忆工具和指标。

你的优势是已经做过机器学习预测模型和 AI 平台搭建，所以不需要把时间花在“AI 是什么”这种浅层内容上。真正要补的是：

- 能不用 Claude 也说清楚：Attention、Embedding、RAG、RLHF、LoRA、BLEU/ROUGE/F1/perplexity。
- 能区分 NVIDIA 生态工具：NeMo、NIM、Triton、TensorRT、RAPIDS、CUDA、NCCL、MIG。
- 能把“做过项目”转成考试语言：数据准备、实验设计、指标选择、部署和可信 AI 风险。
