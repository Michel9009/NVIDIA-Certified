# NVIDIA NCA-GENL 备考资料与学习网站

这个仓库用于备份 NVIDIA-Certified Associate: Generative AI LLMs 认证的两个月备考材料。

## 使用方式

直接打开静态网站入口：

`study-site/index.html`

网站包含：

- 官方考试信息和领域权重
- 8 周日历式学习计划
- 可打卡学习进度
- 知识点地图和考试覆盖矩阵
- 深度概念讲解
- 闪卡和按领域筛选的模拟题
- 测验后弱项反馈

Markdown 版备考资料位于 `prep-materials/`。
